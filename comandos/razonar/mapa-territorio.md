# /razonar:mapa-territorio

Distinguir entre el modelo (mapa) y la realidad (territorio). El mapa NO es el territorio.

## Uso

```
/razonar:mapa-territorio [situación]
```

## Proceso

1. **Identificar** el modelo/mapa que estás usando
2. **Preguntar**: ¿Dónde podría diferir de la realidad?
3. **Buscar** evidencia directa del territorio (datos reales)
4. **Ajustar** el mapa donde sea necesario
5. **Recordar**: todo modelo es una simplificación
