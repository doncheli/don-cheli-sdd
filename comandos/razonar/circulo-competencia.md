# /razonar:circulo-competencia

Conocer tus límites. Distinguir entre lo que conoces profundamente y lo que solo conoces superficialmente.

## Uso

```
/razonar:circulo-competencia [área/decisión]
```

## Proceso

1. **Definir** el área de decisión
2. **Evaluar** honestamente: ¿es mi área de competencia?
3. **Si SÍ**: Decidir con confianza
4. **Si NO**: Buscar experto o investigar antes de decidir
5. **Documentar** los límites para futuras decisiones
