# Decisions

Append-only log of technical and product decisions. Never deleted — only new ones are added or existing ones marked as superseded.

## Format

### DEC-NNN: [Title]
- **Date:** YYYY-MM-DD
- **Status:** active | superseded by DEC-XXX
- **Context:** [Why this decision was made]
- **Decision:** [What was decided]
- **Alternatives:** [What else was evaluated]
- **Consequences:** [What this implies]
