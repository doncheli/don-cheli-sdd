# /razonar:costo-oportunidad

Evaluar el costo de NO elegir las alternativas. Cada decisión tiene un costo oculto.

## Uso

```
/razonar:costo-oportunidad [opciones]
```

## Proceso

1. Listar todas las opciones disponibles
2. Para cada opción: ¿qué GANO y qué PIERDO?
3. Evaluar el costo de la mejor alternativa NO elegida
4. Decidir si el beneficio supera el costo de oportunidad
