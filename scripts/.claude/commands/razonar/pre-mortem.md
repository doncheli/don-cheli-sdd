# /razonar:pre-mortem

Anticipar el fracaso ANTES de que ocurra. Imaginar que el proyecto ya falló y analizar por qué.

## Uso

```
/razonar:pre-mortem [proyecto/decisión]
```

## Proceso

1. **Imaginar** que el proyecto ya falló completamente
2. **Preguntar**: "¿Por qué falló?"
3. **Listar** todas las causas posibles
4. **Evaluar** probabilidad e impacto de cada una
5. **Crear** plan de mitigación para las más probables
